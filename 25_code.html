Code Resources

Brain Wave Component

This headband (wearable device) allows catching brain wave data, specifically EEG information. The information collected by this device is stored in a computer. Brain Wave Component

Heart Rate Component

This is a wearable device which allows the collection of heart rate data, precisely ECG information. The information gathered by this device is stored in the smartwatch memory. Smartwatch Component

Parrot Robot

This component is an object embedded with ubiquitous technology (Raspberry Pi). This object has the color and shape of a parrot, and it emits sounds like a real parrot, so we call it a parrot robot. It has the function of triggering a sound related to an emotional state. Parrot Robot Component

AI

This component contains an AI algorithm. It allows for identifying emotional states in brain waves. It includes an algorithm to identify emotional states in heart rate data. AI Component
